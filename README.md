# 🤖 AI Doubt Solver Chatbot

An AI-powered full-stack chatbot that helps users solve academic queries in real-time using a locally hosted Large Language Model (LLM) via Ollama.

---

## 🚀 Features

- ⚡ Real-time streaming responses (ChatGPT-like experience)
- 🧠 Multi-turn conversations with session-based context
- 📚 Subject-aware query handling
- 💬 Interactive and responsive UI
- 🔄 Fast response optimization using streaming APIs

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express  
- **AI Model:** Ollama (phi3:mini)  
- **Other Tools:** Fetch API, REST APIs
